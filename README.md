# HybridX Car – Smart Hybrid Vehicle

## Overview
HybridX Car is an Arduino-based smart hybrid vehicle designed with three operating modes:
- Remote Control Mode
- Voice Control Mode
- Autonomous Obstacle Avoidance Mode

The project integrates Bluetooth communication and ultrasonic sensing to provide intelligent navigation and seamless mode switching.

## Features
- Remote Control via Bluetooth
- Voice-Controlled Navigation
- Autonomous Obstacle Avoidance
- Intelligent Mode Switching
- Low-Cost Embedded System

## Hardware Used
- Arduino Uno
- HC-05 Bluetooth Module
- HC-SR04 Ultrasonic Sensor
- Arduino uno
- DC Motors
- Chassis
